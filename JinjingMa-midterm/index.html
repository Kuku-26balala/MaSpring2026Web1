Night Sky Animation

This animation is inspired by the night sky. I created a glowing moon using a CSS glow effect to create a calm atmosphere. The title uses a keyframe animation to pop into the screen, making the page feel more dynamic. The navigation bar moves left and right, which I imagined as a spaceship traveling through space. The footer scales up and down like a twinkling star.

Color Selector

This project focuses on interaction. I created five circular buttons with different colors, and each button changes the background when selected. The background images match the color theme, such as a sunflower image for yellow. I added a bounce effect to make the buttons feel more responsive. The layout is simple and clean, and the title uses a keyframe animation.

Ocean Animation

This animation represents an abstract ocean environment. I used an ocean image as the background and added a blue container to resemble an fish tank. The background color changes between different shades of blue to simulate water. Inside the container, simple shapes act as sea creatures. The circle behaves like a jellyfish, and the triangle moves like a fish.
